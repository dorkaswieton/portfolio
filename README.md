In this project, I analyzed the relationship between global temperature anomalies and the frequency of natural disasters using Tableau. ### Project 2: Data-Driven Insights for Smarter Decisions
- Description
In this project, I developed an interactive Tableau dashboard to analyze Airbnb listing data and uncover pricing patterns across neighborhoods and time. Using maps, bar charts, time-series analysis, and KPI metrics, the dashboard helps users explore geographic price differences and seasonal trends to better understand the short-term rental market.
